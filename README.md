### Getting Started

First, clone the repository:

```bash
git clone https://github.com/MdMostaFizurRahaman/microservice-starter.git
```

Then, install the dependencies:

```bash
npm install
```

Then, run the application:

```bash
npm run dev
``` 

Application is running on port 3000.
